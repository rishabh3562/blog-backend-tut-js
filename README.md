# Phase 1: Basic CRUD Operations

Welcome to Phase 1 of the Blog Backend Tutorial! Learn the fundamentals of building a REST API with Node.js, Express, and MongoDB.

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

## API Endpoints

- POST `/users` - Create user
- GET `/users` - Get all users
- GET `/users/:id` - Get user by ID
- PUT `/users/:id` - Full update
- PATCH `/users/:id` - Partial update
- DELETE `/users/:id` - Delete user
